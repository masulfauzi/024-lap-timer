Encryption

Untuk memudahkan Anda dalam melakukan pertukaran informasi yang sifatnya rahasia, Panada telah menyiapkan sebuah class Encryption.

Inisialisasi

Sebelum digunakan, class Encryption harus diinisialisasi terlebih dahulu. Inisialisasi bisa dilakukan pada konstruktor ataupun langsung di dalam method.

$this->crypt = new Resources\Encryption('mySecreteToken');

Berikut adalah contoh penggunaan class Encription:

Tipe Encode

Secara default class Encryption menggunakan PCRYPT_BASE_64 untuk encoding. Anda juga bisa menggunakan tipe lainnya. Pilihannya adalah: PCRYPT_NONE, PCRYPT_BASE_64 dan PCRYPT_HEXA_DECIMAL

$this->crypt = new Resources\Encryption('mySecreteToken', PCRYPT_HEXA_DECIMAL);
